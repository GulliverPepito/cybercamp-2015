Installation
------------

Install url_bruteforcer is so easy:

```
$ python -m pip install url_bruteforcer
```

**Remember that url_bruteforcer only runs in Python 3**.

Quick start
-----------

You can display inline help writing:

```bash

python url_bruteforcer.py -h
```
